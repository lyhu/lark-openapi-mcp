# Skills

This directory contains project-owned Feishu/Lark skills.

Goals:
- Keep the directory structure aligned with `skills-demo/`
- Turn demo skills into project-maintained skills
- Make each skill explicit about which MCP tools it depends on

Conventions:
- One skill per directory
- Each skill directory must contain a `SKILL.md`
- Optional supporting material goes in `references/`
- Prefer real MCP tool names as they are registered by this project
